Prefix: /

	 
Idealistic a simple bot, with features mainly characterizing words, 
with utility commands, yet simple to use.


    Core of Commands
    ================
    
	 
	 
- /help

	 
	 
Displays Avaliable Commands

	 
    
 - /bible

	 
	 
Displays Random Verse from the bible

	 
	 
 - /advise

	 
	 
Shows quotes advises

	 
    
 - /synon [word]

	 
    
Finds synonyms based upon argument


    
 - /haste [words]

	 
    
Saves argument after command, and sends haste link back

	 
    
 - /ss [link]

	 
	 
Screenshots a website based upon address

	 
	 
 - /shorten [link]


	 
Shortens a website link into a is.gd link

	 
	 
 - /flipcoin

	 
	 
Lands on tails or heads
